Panduan Penggunaan
Baca sebelum mengisi form — klik untuk buka
▾
1
Pastikan API sudah dikonfigurasi

Klik tombol ⚙ Konfigurasi API di atas dan masukkan Google Sheets ID serta API Key dari admin kampus. Lakukan sekali saja, data akan tersimpan di browser.

2
Isi identitas diri dengan lengkap

Masukkan nama lengkap, NIM, program studi, dan nomor HP aktif. Pastikan semua data sesuai dengan data akademik.

3
Isi data pengawasan

Pilih nama dosen yang diampu, masukkan tanggal pengawasan, dan tulis keterangan kelas (contoh: "Kelas A – UTS Pemrograman Web").

4
Isi data rekening untuk pencairan honor

Masukkan nomor rekening, nama bank, dan nama pemilik rekening sesuai buku tabungan. Data ini digunakan admin untuk transfer honor.

5
Klik Kirim Laporan

Setelah semua kolom terisi, klik tombol Kirim Laporan. Tunggu konfirmasi hijau bahwa data berhasil dikirim. Tidak perlu kirim ulang jika sudah muncul konfirmasi.

⚠️ Kirim laporan segera setelah selesai mengawas. Data yang sudah dikirim akan diverifikasi admin sebelum honor dicairkan. Jika ada pertanyaan, hubungi admin akademik kampus.
